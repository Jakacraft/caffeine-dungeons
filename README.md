# [CDM] Caffeine Dungeons Mod

## Features

 - Rarity Colors
 - Missing Accessories
 - Better Party Menu
 - Cooldown Hud Display
 - Buffs Hud Display
 - Custom Titles
 - Item Screenshot

## Versions & Dependancies

### Required
 - Minecraft: 1.21.11
 - Fabric: 0.18.0+
 - Fabric Api: 0.141.4+1.21.11
 - Cloth Config: 21.11.153
### Recommended
 - Mod Menu: 17.0.0
